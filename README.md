## Exporting from Blender for Unity

- [Exporting Models from Blender to Unity (YouTube)](https://www.youtube.com/watch?v=kes14A4aSTo)
- [Official Blender to Unity Export Guide](https://docs.unity3d.com/Manual/HOWTO-importObjectBlender.html)
- **Recommended Formats:** `.fbx` and `.obj`  
- Important export considerations:
  - Apply all transforms (`Ctrl+A`)
  - Use correct scale (Unity default is 1 unit = 1 meter)
  - Triangulate meshes as needed
  - Pack textures and materials
